Data Activities using R programming

During the numerical analysis module, we studied twelve units. Every unit had different activities and lectures. For example, we used to read different books to improve our statistical skills. The book that we have been reading the most is Practical Statistics for Data Scientists: 50+ Essential Concepts Using R and Python by Bruce 2020. Also, we improved our programming skills using R during every unit. For example, we used to use the Crime Survey for England and Wales, 2013-2014 dataset to read the data and create boxplot and barplot. Also, we used health data to do some descriptive and inferential statistics and created boxplots and hypotheses. Creating cross-tabulation and chi-square using health data, taking a decision based on p-value, finding a correlation with different variables, and performing simple linear regression analysis.
